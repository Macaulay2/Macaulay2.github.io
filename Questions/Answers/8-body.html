Question 8

If I try to use getPackage to download and install a package whose example code will not run correctly unless a configuration option is set correctly, the examples fail.

Answer

First you must follow the instructions in answer 7 to change the default value of the UserMode option to installPackage from false to true. Then after your first attempt to run getPackage fails, you may set the configuration options by editing the appropriate configuration file (which should be there). Then re-run the getPackage command.
