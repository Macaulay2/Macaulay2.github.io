Downloading Macaulay2 for Cygwin manually

Here is the procedure for downloading the current version (version 1.1) of Macaulay 2 for cygwin manually (when the automatic procedure with setup.exe doesn't work), assuming you've used "setup.exe" to install "wget" and "tar". (For version 1.1 of Macaulay 2, you will need to use setup.exe to install "libgdbm4", "libncurses8", and "libreadline6". For version 1.1.99 or 1.2 of Macaulay 2, install, "lapack", "libgdbm4", "libncurses8", and "libreadline6". For version 1.3, install "lapack", "libgdbm4", "libiconv2", "libncurses9", "libreadline6", "libxml2", and "zlib0". For version 1.3.1, install "lapack", "libgdbm4", "libiconv2", "libncurses9", "libreadline6", "libxml2", and "zlib0".)

    $ cd foo

    $ wget http://www.math.uiuc.edu/Macaulay2/Repositories/cygwin/Macaulay2/Macaulay2-1.1-6.tar.bz2

Running "setup()" after starting Macaulay 2 adds the current location of the Macaulay2 files to your initialization scripts to it can be found. If you install Macaulay2 later in a different location, just run "setup()" again.

Feel free to rename and move the directory "usr" created above to a different location. Run "M2" from the new location and run "setup()" once to record the new location.
